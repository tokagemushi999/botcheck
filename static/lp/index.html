サーバーに潜むBotを、
見つけ出す。

スパムBot、自動投稿アカウント、AI偽装ユーザー。
BotCheckは6つの分析エンジンで不審なアカウントを自動検知し、あなたのサーバーを守ります。

サーバーに追加する 使い方を見る ↓

こんな経験、ありませんか？

BotCheckなし

  • 怪しいアカウントに気づけない
  • スパムが来てから対応するしかない
  • Botかどうか判断に時間がかかる
  • メンバーからの報告頼みで後手に回る

BotCheckあり

  • 不審なアカウントをスコアで可視化
  • リアルタイム監視で問題を未然に防止
  • 6つの分析で根拠ある判定を即座に
  • ダッシュボードでサーバー全体を把握
セットアップ

追加するだけ。設定不要。

1

サーバーに追加

「サーバーに追加する」ボタンからBotを招待。権限は自動設定されます。

2

過去メッセージを自動スキャン

追加した瞬間から、各チャンネルの過去メッセージを自動で取り込み・分析開始。

3

コマンドで分析

/botcheck user で特定ユーザーを分析。/botcheck server でサーバー全体を一覧。

分析エンジン

6つの視点で判定する

タイミング分析
投稿間隔の規則性、応答速度、活動時間帯からBot的なパターンを検出
文体分析
文章の長さ、語彙の多様性、句読点の癖からAI生成テキストを判定
行動分析
チャンネル使用パターン、リプライ率、リアクション傾向から人間らしさを評価
AI検知
定型パターン、不自然な丁寧さ、過度な一貫性などAI特有の痕跡を識別
プロフィール分析
アカウント年齢、アバター、ユーザー名パターンからBot特有の特徴を検出
ネットワーク分析
会話の相互性、メンション関係、孤立度から社会的な人間らしさを評価

無料で使えます。

まずは試してみてください。追加するだけで、すぐに分析が始まります。
