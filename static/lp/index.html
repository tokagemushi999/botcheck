オープンソース
Analysis

1つのスコアに4つの視点

Timing Analysis

投稿間隔の規則性、応答速度、活動時間帯のパターンからBot的な行動を検出。

Style Analysis

文章の長さ、語彙の多様性、句読点の使い方からAI生成テキストの特徴を判定。

Behavior Analysis

チャンネル使用パターン、リプライ率、リアクション行動から人間らしさを評価。

AI Detection

定型文パターン、過度な丁寧さ、不自然な一貫性などAI特有の痕跡を識別。

Real-time Monitoring

全メッセージを自動収集・分析。閾値を超えたユーザーは管理者にアラート通知。

Web Dashboard

サーバー統計、スコア分布、疑わしいユーザーリストをブラウザから確認。

Setup

3ステップで完了

1

Botを招待

「Add to Discord」からサーバーに追加。権限は自動で設定されます。

2

自動スキャン

参加後、各チャンネルの過去メッセージを自動取り込み。進捗はシステムチャンネルに表示。

3

分析開始

/botcheck user でユーザー分析、/botcheck server でサーバー全体を確認。

サーバーを守る準備はできた？

無料で始められます。設定不要、追加するだけ。
